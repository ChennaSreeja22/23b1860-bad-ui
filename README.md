# File Upload From Hell

## Concept

File Upload From Hell is a deliberately frustrating redesign of a normal file upload button.

The upload functionality still works, but the interface actively resists the user through misleading warnings, moving buttons, suspicious messages, and unnecessary interactions. The goal is to parody poor user interface design while keeping the component technically functional.

## Features

* Upload button changes its text randomly
* Button moves away when the user tries to interact with it
* Dragon verification checkbox before uploading
* Fake warnings and misleading information
* Untrustworthy customer reviews
* Judgmental system messages
* Misleading progress bar
* Passive-aggressive success messages

## Why It Is Bad

This design intentionally violates several important UI/UX principles:

* Predictability
* Consistency
* User Control
* Efficiency
* Accessibility
* Clarity of Feedback

The project is intended as a humorous example of how not to design a user interface.
