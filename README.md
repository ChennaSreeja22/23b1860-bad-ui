# File Upload From Hell

## Concept

This project redesigns a normal file upload button as an intentionally frustrating user interface.

The upload functionality still works, but the interface actively resists the user through moving buttons, confusing warnings, misleading feedback, and unnecessary interactions.

## Features

* Button runs away from the cursor
* Random button text
* Dragon verification checkbox
* Fake warnings and reviews
* Judgmental system messages
* Misleading progress bar
* Passive-aggressive success messages

## Why It Is Bad

The design intentionally violates several usability principles:

* Predictability
* Consistency
* User control
* Efficiency
* Accessibility

The goal is to parody poor user interface design while keeping the component technically functional.

* HTML
